== Description ==

Restores the links to the old-fashioned pages for the custom header and the custom background

== Installation ==

= Manual installation =

1. Upload the `tb-old-fashioned-hnb` folder to the `/wp-content/plugins/` directory

= Installation using *Add New Plugin* =

1. From your Admin UI (Dashboard), use the menu to select Plugins -> Add New
2. Click the *Upload plugin* button
3. Select the .zip file and click the *Install now* button

= Activation =

1. Activate the plugin through the *Plugins* menu in WordPress
